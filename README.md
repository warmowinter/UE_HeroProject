修复因武器射击定时器导致的内存泄漏，规范代码格式
测试中发现，在游戏运行时按下鼠标左键，定时器开始运行，如果直接退出时鼠标左键还按着，那么因为鼠标左键松开启用的删除计时器而不起作用，导致游戏退出时，定时器依然在运行。所以添加EndPlay函数在游戏退出时进行删除计时器。
